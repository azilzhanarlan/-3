# food_delivery_analysis

## *Проект: Анализ сервиса доставки еды*

## *Автор: Азильжан Арлан*

## 1. Резюме проекта

Этот проект был создан для анализа данных по доставке еды.
Цель — изучить поведение пользователей, узнать пики спроса и понять закономерности в заказах.

## 2. Бизнес-вопросы, на которые отвечает проект
 1. В какое время суток делается больше всего заказов?
 2. Как расстояние доставки влияет на цену заказа?
 3. Какие часы дня считаются пиковыми?
 4. Как распределяются цены заказов?
 5. Какие полезные инсайты можно получить из графиков?
